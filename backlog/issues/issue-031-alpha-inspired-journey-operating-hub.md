# ISSUE-031: Alpha-Inspired Journey Operating Hub

- Priority: 4-repeatability
- Current classification: future-target
- Evidence count: benchmark plus v0.1.0 repository outputs
- MVP required: no

## Problem

The v0.1.0 repository is a local-first Markdown and template kit. The desired future state is a single Journey operating website with public explanation, private leader resources, current Journey management, role delegation and searchable templates.

## Acceptance Criteria

- Target-state information architecture approved.
- Website content boundaries and copyright rules approved.
- Leader dashboard requirements documented.
- Role assignment and RACI data model agreed.
- Resource library links to consolidated Drive files.
- Prototype validates against privacy, access and governance requirements.

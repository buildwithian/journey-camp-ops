# Leadership Review Agenda

## Purpose

Use this agenda to run the nine scheduled leadership review sessions with explicit checkpoints, prompts, evidence requirements, decision boundaries, expected outputs and time boxes.

## Current Live-state Note

The live `Leadership Review Agenda` tab is populated with the 27 agenda rows from [leadership-review-agenda.csv](leadership-review-agenda.csv), linked into Dashboard row 21, Instructions row 21, Launch Gates evidence fields and Resource Library Index quality checks. Keep the CSV as the recovery/import template.

## Privacy Rule

Record roles, agenda prompts, approved output summaries and links only. Do not enter participant data, medical details, financial account data, private contact details or source-derived names in this agenda.

## Status Values

- Pending session
- Ready to discuss
- In discussion
- Output needed
- Complete
- Waiting for prerequisite
- Waiting for pilot run
- Blocked
- Skipped

## Operating Sequence

1. Start with SCHED-001 agenda rows before assigning dates, reviewers or owner names.
2. Use each agenda row to open one concrete discussion item, check the decision boundary and identify the required output.
3. Record the approved session output in `Leadership Review Outputs` before updating downstream trackers.
4. Keep blocked or waiting rows blocked until the prerequisite approval, review outcome or real pilot evidence exists.

## Local Import Template

The CSV template contains 27 agenda rows:

- AG-001 through AG-003 for launch orientation and review order.
- AG-004 through AG-009 for owner appointment and governance authority review.
- AG-010 through AG-012 for HR-001 through HR-007 reviewer assignment and packet routing.
- AG-013 through AG-018 for pilot planning, pilot debrief and accepted-change routing.
- AG-019 through AG-021 for restricted real-data replacement review.
- AG-022 through AG-024 for website access model review.
- AG-025 through AG-027 for production publication go/no-go review after prerequisites.

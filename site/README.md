# Documentation Site

MkDocs configuration lives at `mkdocs.yml`. Build later with `python3 -m mkdocs build` after installing `requirements-site.txt`.
Primary local entry point: open `site/index.html` to review and use the single-website operating hub concept. The prototype includes Dashboard, Team, Tasks, Events, Library, Gate Review, Access and Drive Copy views; the Gate Review and Access views use synthetic decision rows only and do not publish the site. `site/operating-hub-prototype.html` is retained as the named prototype copy.

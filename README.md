# The Journey Operating System

This repository is the editable, privacy-safe source of truth for a repeatable Journey operating system.

It converts the local Journey source material into structured documentation, templates, validation scripts and a local documentation site. Raw source folders and confidential operational files are intentionally ignored and must not be committed.

Start with [START-HERE.md](START-HERE.md), then review the MVP Run Kit in [MVP-SCOPE.md](MVP-SCOPE.md).

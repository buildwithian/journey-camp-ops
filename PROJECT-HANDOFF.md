# Project Handoff

The repository is a local-first MVP. Raw sources remain ignored in the project folder. The safe release archive is generated under `.work/release/` and excludes raw sources, `.git` and temporary extraction.


## Post-release Direction

The desired future state is an Alpha-inspired target operating hub for The Journey: a single website with public explanation, private leader resources, consolidated templates, role assignments, delegated responsibilities, event readiness and links into a cleaned Google Drive source-of-truth library. See `docs/00-strategy/`.
